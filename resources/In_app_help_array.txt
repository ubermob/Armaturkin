В расчёт можно добавить фоновую арматуру (Вкладка "Ручной ввод", пункт 1)

Пример для шапки таблицы:
Спецификация арматуры на перекрытие на отм. +18,200 (лист 4)

Внимание!
Имя файла не должно содержать следующих символов:
\  /  :  *  ?  "  <  >  |
(Данные символы будут автоматически удалены)
<<>>
Для работы перетащите мышкой нужные файлы (.xlsx) в соответствующие зоны
Файлы в "Каркасах" можно перенаправить на другой вид конструкции, для этого кликните по нужной зоне правой кнопкой мыши. Граница зоны станет в виде штрих-пунктирной линии. Для отмены кликните правой кнопкой мыши по этой зоне ещё раз

В расчёт можно добавить арматуру и изделия из сортамента вручную (Вкладка "Ручной ввод", пункты 2 и 3)

Для тонкой настройки можно использовать .sb файл

Внимание!
Имя файла не должно содержать следующих символов:
\  /  :  *  ?  "  <  >  |
(Данные символы будут автоматически удалены)
<<>>
file .sb
<<>>
Для удаления добавленной записи кликните по ней правой кнопкой мыши
